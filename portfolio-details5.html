About this Project :-

This project analyzes a museum dataset using SQL to gain insights into exhibit popularity, visitor demographics, and revenue generation. It aims to optimize museum operations and enhance visitor experiences.

SQL Functions and Clauses used :-

Joins:

  • To combine the data from multiple tables.

LIMIT Clause:

  • To retrieve a specified number of records.

WHERE Clause:

  • To filter data based on specific conditions.

ORDER BY Clause:

  • Used to sort the data.

GROUP BY Clause:

  • To group the data based on a common attribute.

HAVING Clause:

  • Used to apply a filter on the result of GROUP BY based on the specified condition.

Window Functions:

  • Used to perform calculations against partitions.

Common Table Expressions (CTE):

  • Simplify complex queries and improve query performance.
